First time? Here's the free one-time setup (~5 minutes)
1. Go to firebase.google.com → Console → "Add project" (free, no credit card needed for this).
2. Inside your new project, click "Build → Firestore Database" → Create database → start in test mode.
3. Click "Rules" in Firestore and replace them with this, then Publish: 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /lamp_sync/{code} {
      allow read, write: if true;
    }
  }
}
This keeps it free and simple, but note it's not real authentication — anyone who knows your exact sync code (below) could read or write that one document. Use a long, unguessable code, and don't share it.
4. Click the gear icon → Project settings → scroll to "Your apps" → click the </> web icon → register the app → copy the whole firebaseConfig object it shows you.
5. Paste that config below, pick a sync code (any long phrase only you know), and hit Connect. Do the exact same on your other device.
Without sync connected, everything stays only in this browser (no account, no tracking).
